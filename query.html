Query suggester

Description:

Query suggester is a web app developed with Node.js. Its purpose is to suggest tables, columns and queries to finally have a result from the database. All its sugestions are according to the tables that you have in your database. The results are displayed on bootstrap tables. You can select, insert and update tables.

Technologies: Javascript, bootstrap, html, css, Node, Express, Mysql
